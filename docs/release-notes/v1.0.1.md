# Veyra Node.js SDK v1.0.1

Release date: 2026-05-17

## Highlights

- Adds first-class SDK support for the backend's updated Responses API reasoning fields.
- Expands the public TypeScript contracts for reasoning output, cached token usage, reasoning token usage, assistant responses, billing records, quota limits, and image response formats.
- Fixes raw-response typing so `client.withRawResponse.*` methods expose `APIResponse<T>` metadata while preserving strongly typed data.
- Replaces minimal docs and examples with deeper usage guides and runnable workflows.
- Adds regression tests around the SDK behavior most likely to break across backend updates.

## Compatibility

This is a backward-compatible patch release. Existing non-streaming and streaming chat, completions, embeddings, images, audio, models, quota, billing, API key, assistant, and health methods remain available under the same namespaces.

The Responses API may return reasoning output items before message output items. Applications should search `response.output` for `item.type === "message"` before reading text.
